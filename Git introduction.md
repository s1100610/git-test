# Introduction of Git

## How to use Git
- Right click on the folder and select "Open with code"
- Open terminal(Ctrl+Shift+@)
- $ Git init
- $ git add .
- $ git commit -m "comment"
- $ git remote add origin "URL"
- $ git push origin master

- Ctrl+Shift+@ -> Open terminal
